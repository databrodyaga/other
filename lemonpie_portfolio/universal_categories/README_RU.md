# Классификация платежей по универсальным категориям

В данном разделе добавляем признак универсальных категорий, сформированный на базе классификации Минюста и рекомендованной рынком практики, применяем методы предобработки и разметки данных, обучения набора моделей для классификации входящих платежей фондов-клиентов.

Этапы:
- данные платежей загружаются по API в аналогичном прогнозированию входящих платежей формате (request_3.sql), но дополненном названиями статей, проектов и доноров (request_4.sql), а также меток uc(request_5.sql) c параметром incoming;
- uc_data_labeling - разметка платежей по универсальным категориям по согласованному списку (универсальные категории_3_1.docx/articles_uc.csv/articles_uc_added.csv) и частично вручную;
- gpt_classification - тестирование классификации универсальных категорий по прогнозам публичных LLM - ChatGPT 5/Gemini 2.5 pro/YandexGPT и оценка метрик.  
- own_classification - создание и тестирование собственных моделей классификации платежей по универсальным категориям.
