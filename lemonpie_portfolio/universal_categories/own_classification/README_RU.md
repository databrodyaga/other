# Классификация универсальных категорий с применение собственной ML-модели

В данном разделе анализируем данные для прогнозирования меток, тестируем разные пайплайны обработки данных и настраиваем прогнозные модели.

В папке:
- check_date_wo_forecasts.ipynb - наличие меток после ручной разметки тренировочного датасета (все платежи по 09.10.2025 включительно)
- classify_manual.ipynb - ручная загрузка меток в БД по замечаниям эксперта/клиентов
- EDA_for_classify.ipynb - анализ данных и кореляций между признаками
- prod - скрипт для классификации платежей, production-версия модели и дополнительных данных
- train_test - пайплайны подбора моделей и предобработки данных, тесты моделей/признаков, переобучение production-модели
