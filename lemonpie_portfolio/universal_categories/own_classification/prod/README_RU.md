# Production-версия модели и скрипт для классификации платежей

В папке:
- classify_forecast_prod (ipynb/py) - jupyter/python - скрипт загрузки новых платежей, генерирования прогнозов и отправки меток в БД
- classify_forecast_prod.log - лог отработки скрипта
- model_cb.cbm - catboost-модель для генерирования прогнозов
- ipca_*.pkl, scaler_*.pkl - вспомогательные данные для генерирования признаков для модели из текстовых полей исходных данных, данные представления генерируются при (пере)обучении модели
