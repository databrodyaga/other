# Анализ и прогнозирование данных платежей

Анализируем данные по платежам от донорров в адрес фондов, прогнозируем суммы платежей в различных формах - каждый из платежей в отдельности по описательным признакам, суммы платежей суммарно на следующей неделе, следующем месяце, а также на несколько временных отрезков вперед, также с группировкой по различным описательным признакам (фонды, провайдеры платежей, тип подписки, категории платежей).

В **first-try-origin.ipynb** - данные подготовлены для прогнозирования сумм исходя из описательных признаков - без учета временной составляющей и группировки по признакам, общая модель на всех данных, которая прогнозирует сумму платежа. Также в этом файле проведен анализ данных (дополняется).

В **first-try-sarimax.ipynb** данные подготовлены для прогнозирования сумм на следующий день методом SARIMAX (для временного ряда) как промежуточный вариант, который будет использоваться для генерации доппризнака при обучении модели на базе catboost.

В **first-try-cb_sarimax.ipynb** данные подготовлены для прогнозирования сумм исходя из описательных признаков c учетом временной составляющей (прогноза SARIMAX как дополнительный признак) и группировки по признакам.

В **first-try-3-models.ipynb** данные подготовлены для прогнозирования сумм исходя из описательных признаков c учетом временной составляющей на базе 3 моделей(LSTM, Sarimax,  Catboost) и взвешивания прогнозов несколькими способами.

Данный раздел будет дополняться.

**Описание данных**:  

fund_id: это ID фонда  
full_sum: сумма платежа  
date: дата в unixtimestamp  
provider: ID провайдера, через который сделан платеж  
signup: тип подписки: 0 - разовый платеж, 1 - регулярный платеж  
category_1, category_2, category_3, category_4, category_5, category_6, category_7: 0 - НЕ относится к этой категории, 1 - относится к этой категории
