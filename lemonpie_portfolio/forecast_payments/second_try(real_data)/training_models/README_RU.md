# Обучение моделей и production-скрипт

Рефакторим python-скрипт из ipynb-нотбука EDA/3_feature_selection.  
Предобрабатываем данные и обучаем модели для прогнозирования поступлений платежей.
Текущий скрипт **second_try.py** - фактически продакшн модель для ежедневных прогнозов.

Файлы в папке:
 - **second_try.py** - итоговый скрипт с пайплайном на 3 модели (с временной составляющей)
 - **request_5.sql** - запрос к базе (набор сырых данных **data_*.csv** не загружен из-за размера)
 - **users_actual.csv** - действующие user_id (фонды) для фильтра сырых данных, на текущем этапе не используется
