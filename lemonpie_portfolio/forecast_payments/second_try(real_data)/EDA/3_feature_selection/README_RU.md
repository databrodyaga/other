**Сравнение day_purpose_mean и 5 PCA-компонент**:  
*CatBoost*: PCA-компоненты (особенно первые 1-3) часто показывают более высокую индивидуальную и совокупную важность, чем day_purpose_mean.   
*LSTM*:  
day_purpose_mean: влияние нестабильно, иногда полезен, иногда его удаление улучшает модель.  
PCA-компоненты: Также нестабильное влияние по отдельности. Первые компоненты (pca_1, pca_2, pca_3) чаще показывают себя важными (их удаление ухудшает метрику для некоторых фондов), в то время как pca_4, pca_5 чаще можно удалить без вреда или даже с поьлзой.
**Вывод**:
Набор из нескольких PCA-компонент (например, первых 3-х) вероятно, будет более информативным, чем один day_purpose_mean. day_purpose_mean можно оставить в дополнение - он не сильно коррелирует с PCA-компонентами и добавляет свою уникальную (хоть и небольшую) важность.

На основании выводов выше был сформирован пул признаков к исключению - feature_set_2 в файле comparison. Дополнительно были протестированы иные наборы признаков с расчетом метрик на тестовом наборе по одному и тому же набору фондов. 
Данные приведены в файле **comparison** на вкладках с соответствующими метками: 
- указаны наборы исключаемых признаков, 
- абсолютные значения метрик,
- нормализованные (на минимальное значение одной из моделей для данного фонда) значения метрики RMSE, 
чтобы оценить отдельные модели и ансамбли между собой , а усредненные значения по всем фондам позволяют сравнить результаты прогнозов на разных наборах признаков (между вкладками и свод на вкладке comparison).  

Также приведены аналогичные данные по самым эффективным наборам признаков для топ-10 фондов по количеству данных (последняя вкладка - comparison_normalized).

**В итоге** набор feature_set_2 на базе отсева по важности и по-очерЕдному обнулению признаков показал один из лучших вариантов, немного доработанный вариант набора - **feature_set_6** был выбран итого для применения на продакшене (чуть меньше признаков, исключение которых позволить упростить алгоритм обучение + возвращен один из признаков, показывающий стабильный прирост метрики).
**Между разными моделями и ансамблями**, периодические выбросы эффективности Catboost на отдельных фондах - в среднем проигрывают ансамблю на базе трех моделей с взвешиванием коэффициентов простой нейросеткой (3models_ens_nn_rmse).
