# Сравнение и отбор пайплайнов

Предобрабатываем данные и формируем два пайплайна с 2мя (без временной сотавляющей) и 3мя (с временной составляющей) моделями для прогнозирования поступлений платежей.  
В файле сравнения метрик видно, что более полный пайплайн с тремя моделями и достроенным временным рядом значительно превосходит простой вариант, оставляем более сложную версию.

Файлы в папке:
 - **second_try_2_models.ipynb** - ноутбук с пайплайном на 2 модели (без временной составляющей)
 - **second_try_3_models.ipynb** - ноутбук с пайплайном на 3 модели (с временной составляющей)
 - **comparison_.csv**, **comparison_.xlx** - сравнение метрик двух пайплайнов (см.выше) по имеющимся фондам
 - **request_3.sql** - запрос к базе (набор сырых данных **data_3.csv** не загружен из-за размера)
 - **users_actual.csv** - действующие user_id (фонды) для фильтра сырых данных
 - **2try_data_with_purpose_mean.parquet** - кэш-файл с расчетами эмбеддингов языковых моделей для кодирования текстовых полей.
