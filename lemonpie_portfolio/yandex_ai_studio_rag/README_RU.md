# Настройка LLM с RAG для подключения к чату техподдержки LemonPie

В данном разделе размещены скрипты генерации запросов, обработки текстовых массивов для корпоративной базы знаний, настройки и управления поисковыми индексами по базе знаний:  

- **rag_ver_1** - первая генерация на основе библиотеки YandexCloudML;
- **rag_ver_2** - вторая генерация на основе библиотеки Responses API(OpenAI).
