# Настройка LLM с RAG через YandexCloudML

Изначальная версия подключения предполагала предобработку текстов для базы знаний из расшифровок обучающих видео и мастерклассов в формат markdown по схеме "вопрос-ответ", размещение файлов в хранилище на сервере, построение поискового индекса и настройку AI-агента с подключением нструментов поиска по базе знаний и запросами в Интеренет при необходимости. Корпоративный чат, получив запрос от пользователя из чата - направлял его AI-агенту, и транслировал ответ.  
База знаний регулярно пополнялась экспертами компании.

Состав папки:
- **.md, .jsonl** - файлы базы знаний;  
- **zapros_rag.py** - скрипт запроса;
- **transform_jsonl_md.py** - скрипт трансформации текстов/json в markdown формат по схеме "вопрос-ответ".
